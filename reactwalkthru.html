React and You!

React is a front facing UI for modern web development. We're using it in our project so this document will detail how to use it and when.



Setup

  • Node.JS
  • NPM
  • Access to terminal/bash

After you've gathered and installed all the components listed above. You can open a terminal and navigate to your folder.

Run this code to initiate the Node Package Manager for your project

npm init

Run the following to install create-react-app as a dependency to your project.

C:\users\you> npm install create-react-app -s

Run the following code to initiate the create-react-app package

C:\users\you> npx create-react-app [name of app]

Running the app

From here you should have a working app. To initialize and view it in a browser run the following command.

C:\users\you> cd [name of app]

then

C:\users\you\name-of-app> npm start
